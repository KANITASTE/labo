① 金属 ＋ 水 — Metal + Water

試薬A試薬B化学反応式観察される変化種別
カリウム K水 H₂O2K + 2H₂O → 2KOH + H₂↑H₂↑ 激しく発火・水面を激しく走る・紫の炎酸化還元
ナトリウム Na水 H₂O2Na + 2H₂O → 2NaOH + H₂↑H₂↑ 発火・水面をスキター・黄炎・pH↑酸化還元
カルシウム Ca水 H₂OCa + 2H₂O → Ca(OH)₂ + H₂↑H₂↑ 気泡・白濁（Ca(OH)₂）・pH↑酸化還元

② 金属 ＋ 塩酸（HCl） — Metal + Hydrochloric Acid

金属試薬化学反応式（代表）観察される変化備考
マグネシウム Mg塩酸 HClMg + 2HCl → MgCl₂ + H₂↑H₂↑ 激しい気泡・Mg溶解反応性大
アルミニウム Al塩酸 HCl2Al + 6HCl → 2AlCl₃ + 3H₂↑H₂↑ 気泡・Al溶解両性金属
亜鉛 Zn塩酸 HClZn + 2HCl → ZnCl₂ + H₂↑H₂↑ 気泡・Zn溶解—
鉄 Fe塩酸 HClFe + 2HCl → FeCl₂ + H₂↑H₂↑ 気泡・淡緑色Fe²⁺—
ニッケル Ni塩酸 HClNi + 2HCl → NiCl₂ + H₂↑H₂↑・緑色Ni²⁺生成ゆっくり反応
スズ Sn塩酸 HClSn + 2HCl → SnCl₂ + H₂↑H₂↑・Sn²⁺生成両性金属
クロム Cr塩酸 HClCr + 2HCl → CrCl₂ + H₂↑H₂↑・青色Cr²⁺濃硝酸では不動態
コバルト Co塩酸 HClCo + 2HCl → CoCl₂ + H₂↑H₂↑・桃色Co²⁺—
マンガン Mn塩酸 HClMn + 2HCl → MnCl₂ + H₂↑H₂↑ 気泡・Mn²⁺生成反応性大
ナトリウム Na塩酸 HCl2Na + 2HCl → 2NaCl + H₂↑H₂↑ 激しい・発火—

③ 金属 ＋ 希硫酸（H₂SO₄ 希）— Metal + Dilute Sulfuric Acid

金属試薬化学反応式観察される変化
鉄 Fe希硫酸Fe + H₂SO₄ → FeSO₄ + H₂↑H₂↑ 気泡・淡緑色FeSO₄溶液
ニッケル Ni希硫酸Ni + H₂SO₄ → NiSO₄ + H₂↑H₂↑・緑色NiSO₄
銅 Cu希硫酸— 反応なし —変化なし（イオン化傾向 Cu < H）

④ 金属 ＋ 希硝酸（HNO₃ 希）— Metal + Dilute Nitric Acid

金属試薬化学反応式観察される変化
銅 Cu希硝酸3Cu + 8HNO₃(希) → 3Cu(NO₃)₂ + 2NO↑ + 4H₂ONO↑ 無色（空気中で赤褐色NO₂へ）・溶液が青色Cu²⁺
銀 Ag希硝酸3Ag + 4HNO₃(希) → 3AgNO₃ + NO↑ + 2H₂ONO↑・Ag溶解・Ag⁺生成
金 Au希硝酸— 反応なし —金は希硝酸に溶けない（王水のみ）

⑤ 金属 ＋ 濃硝酸（HNO₃ 濃）— Metal + Conc. Nitric Acid

金属試薬化学反応式観察される変化
銅 Cu濃硝酸Cu + 4HNO₃(濃) → Cu(NO₃)₂ + 2NO₂↑ + 2H₂ONO₂↑ 赤褐色・有毒・溶液が青色Cu²⁺
銀 Ag濃硝酸Ag + 2HNO₃(濃) → AgNO₃ + NO₂↑ + H₂ONO₂↑・Ag溶解
水銀 Hg濃硝酸Hg + 4HNO₃(濃) → Hg(NO₃)₂ + 2NO₂↑ + 2H₂ONO₂↑・Hg溶解
クロム Cr濃硝酸（不動態化）反応停止・不動態被膜形成・H₂発生なし
金 Au濃硝酸— 反応なし —金は濃硝酸にも溶けない

⑥ 濃硫酸（熱）・王水 — Conc. H₂SO₄ (hot) / Aqua Regia

試薬A試薬B条件化学反応式観察される変化
銅 Cu濃硫酸加熱必須Cu + 2H₂SO₄(濃) →(熱) CuSO₄ + SO₂↑ + 2H₂OSO₂↑ 刺激臭・有毒・溶液が青色
銅 Cu希硫酸常温— 反応なし —変化なし
金 Au塩酸 + 濃硝酸（王水）常温Au + 4HCl + HNO₃ → [AuCl₄]⁻ + NO↑ + 2H₂ONO↑・黄色[AuCl₄]⁻・金が溶解

⑩ 金属置換反応（イオン化傾向） — Metal Displacement

イオン化傾向の大きい金属が、より小さい金属をイオンから置き換える。
K > Ca > Na > Mg > Al > Zn > Fe > Ni > Sn > Pb > H > Cu > Hg > Ag > Pt > Au
金属溶液イオン反応式観察される変化
亜鉛 Zn硫酸銅 CuSO₄Zn + Cu²⁺ → Zn²⁺ + Cu↓赤銅色の銅析出・青色が薄まる（Zn > Cu）
鉄 Fe硫酸銅 CuSO₄Fe + Cu²⁺ → Fe²⁺ + Cu↓赤銅色の銅析出・緑色Fe²⁺生成（Fe > Cu）
銅 Cu硝酸銀 AgNO₃Cu + 2Ag⁺ → Cu²⁺ + 2Ag↓銀樹析出・溶液が青色Cu²⁺（Cu > Ag）

⑬ 反応しない組み合わせ（教育的ポイント）— Non-reactions

試薬A試薬Bなぜ反応しないか
銅 Cu希硫酸イオン化傾向 Cu < H → Cu はH⁺を還元できない
銅 Cu塩酸 HCl同上（非酸化性酸には溶けない）
金 Au希硝酸・濃硝酸・塩酸・希硫酸Au は最もイオン化傾向が小さく王水のみに溶解
クロム Cr濃硝酸不動態：緻密な酸化被膜が形成され反応停止
銀 Ag塩酸 HCl / 希硫酸Ag は酸化性酸（硝酸）以外には実質溶けない
